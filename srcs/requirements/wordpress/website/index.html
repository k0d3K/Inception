Inception

A Docker-based project to set up a fully functional WordPress website with Nginx, MariaDB, and PHP-FPM ‚Äî as required by the 42 Inception project.

üß© Overview

This project automates the deployment of a WordPress site using Docker containers. It includes:

  • Nginx: Web server with HTTPS enabled via a self-signed certificate.
  • MariaDB: Lightweight and fast database server.
  • WordPress: Content management system.
    • PHP-FPM: To serve dynamic content.
    • WP-CLI: For headless WordPress configuration.

Everything is orchestrated via docker-compose and managed using a Makefile.

üß† Quick explanations

What is Docker?

Docker lets you package and run applications in isolated environments called containers. Each container contains everything it needs to run ‚Äî making deployment reliable and portable.

Containers used in this project

  • Nginx: The web server that handles requests and manages HTTPS.
  • MariaDB: Stores your WordPress content and settings.
  • WordPress with PHP-FPM: Runs WordPress logic, generates pages, and connects to MariaDB.
  • Redis (bonus): In-memory cache to speed up the site.
  • FTP server (bonus): Lets you transfer files in/out of the site.
  • Adminer (bonus): Browser-based tool to manage the database.
  • Glances (bonus): Real-time monitoring dashboard for your stack.
